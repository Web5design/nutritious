Tagz
====
Tagging app for all kinds of things (books, quotes, persons).


Features
--------
* Scoped search (with q=)
  * Even regex support (encode + as %2b)
    * http://127.0.0.1:8000/tagz/lib/NASB/Luke?q=hi\w%2bt
* Highlighted Context
* Tags library
* Different kinds of books (books, bible, quotes)
